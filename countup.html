countup is an htmlwidget for countup.js It is obviously meant to be used in Rmarkdown html documents (such as this very website).

code Repository

Install

devtools::install_github("JohnCoene/countup")

Examples

library(countup)

countup(25) # basic examples

opts = list(
  useEasing = TRUE, 
  useGrouping = TRUE, 
  separator = ',', 
  decimal = '.', 
  prefix = 'US$', 
  suffix = ' bejillion' 
)

countup(count = 1729, start = 10, options = opts) # with options
