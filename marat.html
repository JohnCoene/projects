Marat


Revive the values of the Enlightenment with marat!


code Repository devices Demo file_download Download

Marat is heavily inspired by L'Ami du peuple , a newspaper written by Jean Paul Marat during the French Revolution, in which he was vocal advocate for the rights of man and liberty.

  1. Adapt the _config.yml file.
  2. Replace/Delete the posts.
  3. Change about.md
  4. Change or add your links in the nav.yml file located in the _data folder.
  5. Enlighten the masses

Run with:

bundle exec jekyll serve --watch
