Rmarkdown

Being an htmlwidget, chartist is compatible with all HTML Rmarkdown formats: this very website is made with Rmarkdown.

Example

---
title: "chartist example"
author: "John doe"
date: "`r Sys.Date()`"
output: html_document
---

```{r setup, include=FALSE}
knitr::opts_chunk$set(echo = TRUE)
library(chartist)
```

```{r}
mtcars %>%
  chart_ist(qsec) %>%
  add_ist(mpg) 

```

Options

Default knitr options:

  • width = 100%
  • height = 400px

You can override that by specifying width and/or height in chart_ist.
